K-Means Clustering Algorithm** on the Iris dataset using two features:
- Sepal Length  
- Sepal Width  
The dataset is divided into **3 clusters** using unsupervised learning techniques.
The Iris dataset contains:
- 150 flower samples
- 3 species (Setosa, Versicolor, Virginica)
- 4 numerical features
Objective
- Apply K-Means clustering
- Divide the dataset into 3 clusters
- Visualize cluster separation
- Analyze grouping pattern
